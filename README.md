# SportradarTask

This is my solution to Sportradar backend task.

App was written in Spring with Thymeleaf.

To run this app open SportradarTaskApplication.java file and click run. Later on, do what you want depending on what is shown on the screen.

I prepared my ERD based on file (DATA. JSON file.pdf) attached in your invitation email. So the whole tables and relations look good, but some tables are left unused because of no need in task specification.

'Raw mode' was useful for me during the first steps of development. Later, I developed also a front-end, but I decided to keep also this 'raw mode' to not get rid of work that I has already done.

Log informations gives an insight of what is going on.
