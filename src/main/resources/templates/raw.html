Hello Sportradar!

This is an example usage with no UI



First, add two teams:


POST /save-team → to add new team

Now, that you have teams, you can add a match:

POST /save-match → to add new match


{
   "match": {
      "discipline": "Football",
      "season": "2019",
      "status": "PLAYED",
      "dateVenue": "18.07.2019",
      "dayOfWeek": "THURSDAY",
      "timeVenueUTC": "18:30",
      "stadium": null,
      "groupCol": null
   },
   "homeTeamName": "Salzburg",
   "awayTeamName": "Sturm"
}

Now, that you have teams and match, you can retrieve them with the following endpoints:

GET /find-all-matches → to get all matches

GET /find-one-match/{id} → to get single match by its id

We can also retrieve teams:

GET /find-all-teams → to get all teams

GET /find-one-team/{name} → to get single team by its name
