Raw Mode



First, add two coaches:


POST /save-coach-raw → to add new coach



{
   "firstName": "Daniel",
   "lastName": "Beichler"
}
and

{
   "firstName": "Fabio",
   "lastName": "Ingolitsch"
}

Then, add two teams:


POST /save-team-raw → to add new team



{
   "team": {
      "name": "Salzburg",
      "officialName": "Salzburg",
      "slug": "salzburg",
      "abbreviation": "RBS",
      "countryCode": "AUT",
      "stagePosition": null,
      "yearOfFoundation": "1933"
   },
   "coachFirstName": "Daniel",

and

{
   "team": {

      "officialName": "Sturm",

      "abbreviation": "STU",
      "countryCode": "AUT",
      "stagePosition": null,
      "coach": "Fabio Ingolitsch",
      "yearOfFoundation": "1909"
   },
   "coachFirstName": "Fabio",
   "coachLastName": "Ingolitsch"

Now, that you have teams, you can add a match:

POST /save-match-raw → to add new match


{
   "match": {
      "discipline": "Football",
      "season": "2019",
      "status": "PLAYED",
      "dateVenue": "18.07.2019",
      "dayOfWeek": "THURSDAY",
      "timeVenueUTC": "18:30",
      "stadium": null,
      "groupCol": null
   },
   "homeTeamName": "Salzburg",
   "awayTeamName": "Sturm"
}

Now, that you have match, you can retrieve it with the following endpoints:

GET /find-all-matches-raw → to get all matches

GET /find-single-match-raw/{id} → to get single match by its id

We can also retrieve teams:

GET /find-all-teams-raw → to get all teams

GET /find-single-team-raw/{name} → to get single team by its name

And coaches:

GET /find-all-coaches-raw → to get all coaches

GET /find-single-coach-raw/{firstname}/{lastname} → to get single coach by its name
